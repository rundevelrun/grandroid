**Endroid** is a free Android emulator for Windows built for gamers and power users.
Game controls, macro recording, multi-instance support — all built-in. No subscription required.

---

## Features

- **Game Controls** — Map keyboard & mouse to touch. Save presets per game.
- **Macro Recording** — Record and replay touch sequences. Speed multiplier, loop support.
- **Multi-Instance** — Run multiple Android instances simultaneously.
- **Fast Performance** — Hyper-V acceleration, GPU passthrough, scrcpy display (~35ms latency).
- **APK Drag & Drop** — Drop an APK onto the window to install instantly.
- **Play Store** — Install Google Play Store in one click (no account required).
- **Android 15** — Latest Android version with full app compatibility.
- **Free** — Ad-supported. No subscription, no account.

## Getting Started

1. Download and run the installer
2. Hyper-V is enabled automatically
3. On first launch, the Android image is downloaded from Google's servers (~1GB)
4. Create an instance and start using Android on your PC

---

## System Requirements

- Windows 10 version 2004 or later (Windows 11 recommended)
- 64-bit processor with virtualization support
- 8GB RAM minimum (16GB recommended)
- Hyper-V compatible hardware
