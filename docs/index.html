Everything you need, nothing you don't

Game Controls

Map keyboard and mouse to touch controls. Save presets per game. Works with any Android game out of the box.

Macro Recording

Record and replay touch sequences with speed multiplier and loop support. Automate repetitive in-game actions.

Multi-Instance

Run multiple Android instances simultaneously. Each instance is isolated with its own storage and ports.

Fast & Lightweight

Powered by Hyper-V and hardware GPU passthrough. Low latency screen mirroring via scrcpy (~35ms).

APK Drag & Drop

Install APK files by dragging them onto the window. Automatically installs to all running instances.

Play Store Support

Install Google Play Store in one click. No configuration needed — just follow the on-screen guide.
